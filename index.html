Preguntas Frecuentes

El DOM (Document Object Model) es una representación en árbol de tu HTML que JavaScript puede leer y modificar.

Es una función que "escucha" o espera a que ocurra algo específico (como un clic) para ejecutar un código.

Úsalo cuando necesites guardar datos pequeños (como un carrito o tema oscuro) que persistan aunque cierres el navegador.
